This repository contains my personal solutions for the [PortSwigger Web Security Academy](https://portswigger.net/web-security). 

I try to explain my way of thinking, approaching, and solving the issue at hand. Treat them more like a scratchpad, not a proper, full report on each lab.

The repository is split into Paths and Labs - as labs are an essential part of the learning paths at the Academy, but not all of them are used. The repository is best viewed using Quartz or Obsidian to support features like note embedding, which GitHub does not natively render.

I primarily use Burp Suite Community Edition. Any addons I also use will be mentioned in the notes. While some labs are designed with Pro features in mind (like the Collaborator), I note any workarounds I used to achieve a "Solved" status. Simple labs with self-explanatory titles may contain less detail.
